# Welcome, or: why this site exists

I'm a few weeks out from graduating NC State, which means I'm in that stretch of
the semester where you suddenly remember you've been meaning to build a personal
site for about two years. So here it is.

## What this is

Mostly a home for my projects and a slightly-longer version of my resume. If
you're reading this because you're hiring, the **Projects** and **About** pages
are probably what you want. If you're reading this because you know me, hi.

## What I'll probably write here

No real promises, but likely:

- **Build logs** from the Juntos capstone and whatever comes next.
- **Short notes** on things I'm figuring out: Spring Boot, Next.js, Docker
  weirdness, whatever's currently on fire.
- **Less serious stuff**, because not everything has to be a LinkedIn post.
  Grill reviews. Elden Ring runes. Woodworking failures. We'll see.

If you have opinions about any of the above, [drop me a line](/contact).
